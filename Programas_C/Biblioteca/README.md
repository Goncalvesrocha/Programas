Sistema de Gerenciamento de Biblioteca em C

Este é um projeto feito em C para gerenciar uma biblioteca de forma simples. Ele utiliza structs, ponteiros e alocação dinâmica de memória, mostrando conceitos essenciais da linguagem.
O sistema roda no console, com um menu interativo, permitindo controlar livros, leitores e empréstimos de forma prática.

⚙️ Funcionalidades

📖 Livros

Cadastrar novos livros

Buscar livros por código

Listar todos os livros cadastrados

Remover livros do sistema


👤 Leitores

Cadastrar leitores

Buscar leitores cadastrados

Listar todos os leitores

Remover leitores do sistema


🔁 Empréstimos

Registrar empréstimos de livros

Controlar datas de empréstimo e devolução prevista

Registrar devolução de livros

Listar todos os empréstimos ativos


🧠 Conceitos de C aplicados

Este projeto é ótimo para treinar:

struct para organizar os dados

Uso de ponteiros

Alocação dinâmica (malloc, calloc, realloc)

Manipulação de strings (string.h)

Manipulação de datas (time.h)

Organização de código em funções

Menus interativos no terminal

Lógica de sistemas CRUD (Criar, Ler, Atualizar, Excluir)
